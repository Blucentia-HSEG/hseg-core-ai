## Key Features Documented

### ü§ñ AI/ML Components
- **Multi-Domain Assessment**: Healthcare, Schools, Enterprise, Government sector analysis
- **Individual Risk Prediction**: Ensemble model for personal risk assessment with HSEG tier classification
- **Text Crisis Detection**: NLP-based crisis intervention identification with Work War integration
- **Organizational Aggregation**: Enterprise-level culture health metrics with social pressure protocols
- **HSEG Scoring System**: 28-point standardized scoring with five-tier classification (Crisis to Thriving)

## Architecture Decision Records

Key architectural decisions documented in this repository:
- **Microservices over Monolith**: Scalability and maintainability
- **FastAPI over Flask**: Performance and automatic documentation
- **PostgreSQL over MongoDB**: ACID compliance and complex queries
- **Kubernetes over VM Deployment**: Container orchestration and scaling
- **React over Vue**: Component ecosystem and TypeScript support
